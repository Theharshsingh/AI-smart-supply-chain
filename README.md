
# SupplyChain Guardian
**Preemptive Disruption Detection & Route Optimization System**

## 🎯 Problem Statement
Modern supply chains face critical transit disruptions (weather, bottlenecks) detected too late, causing cascading delays. We build a scalable system to **predict disruptions early** and **optimize routes dynamically**.

## 🚀 Solution Overview
Real-time analysis of multimodal transit data (GPS, weather, traffic, IoT) to:
- Detect anomalies before they cascade
- Recommend optimal route adjustments instantly
- Minimize delivery delays proactively

Frontend:  React 19, Vite, Tailwind CSS, React Leaflet (Leaflet.js), Recharts, Framer Motion, Socket.io-client
#Backend: Node.js, Express.js, Socket.io, node-cron, JWT (jsonwebtoken), bcryptjs
#APIs & Integrations: Google Maps API (routing, geocoding), OpenWeatherMap API (weather intelligence)
#Tools & Libraries: Axios / Fetch, UUID, dotenv, Lucide React, React Hot Toast, QRCode.react
#Concepts: Real-Time Systems, WebSockets, Heuristic Optimization, Predictive Routing, Simulation Systems
